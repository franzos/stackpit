Setup

Use sentry-cli to upload source maps. Set these environment variables:

export SENTRY_URL={{ sentry_url }}
export SENTRY_AUTH_TOKEN=<your API key>
export SENTRY_ORG=default
export SENTRY_PROJECT={{ project_id }}

Then upload:

sentry-cli sourcemaps upload --release=<version> ./dist
